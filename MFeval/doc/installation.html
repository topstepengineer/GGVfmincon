Downloading

MFeval can be downloaded from the MATLAB File Exchange.

Installing & Uninstalling

To install the toolbox you need to open the .mltbx file downloaded from the MATLAB File Exchange. You can do this by double clicking it or by dragging and dropping it in to the command window. Once installed the toolbox will be available from any working directory.

If you are running a MATLAB version prior to R2014b you will have to download the .zip file, extract the content in a folder and add that folder to the search path manually using the Set Path button. .

You can uninstall the toolbox by clicking on the Manage Add-Ons menu in the Home toolbar, as shown below and then removing MFeval from your list of installed tools.

Running

Once you have installed MFeval you can run the function in any path. Type 'help mfeval' in the command window for more information.

System Requirements

  • MATLAB R2017b or above.
